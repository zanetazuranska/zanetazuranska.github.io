Custom 3D Engine (C++ / OpenGL)

Currently, I am working together with a team on a custom 3D game engine built in C++ using OpenGL. The engine is being developed as a foundation for our own game project, focusing on scalability, modularity, and performance.

My main responsibilities include development of core engine systems such as the asset manager, rendering module, editor, and an intelligent enemy AI system.

Key Responsibilities

  • Asset management system (loading, organization, and lifecycle handling)
  • Rendering module (OpenGL-based rendering pipeline)
  • In-engine editor development
  • AI system for intelligent enemy behavior

Project Context

The engine is being developed as part of a larger game project, allowing us to build full control over core systems and deeply understand low-level game architecture.

Events & Showcase

The project will be presented at ZTGK in Łódź in June, where we will showcase both the engine and the game built on top of it.
